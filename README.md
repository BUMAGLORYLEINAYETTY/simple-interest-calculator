# Simple Interest Calculator

This project contains a Bash script that calculates simple interest based on user input.

## Project Details

The Simple Interest Calculator is a command-line tool that computes interest using the formula:

**Simple Interest = (Principal × Rate × Time) / 100**

## Input Fields

The script requires three inputs from the user:
- **Principal (P)** - The initial amount of money
- **Rate of Interest (R)** - The annual interest rate in percentage
- **Time Period (T)** - The duration in years

## Formula Explanation

Simple interest is calculated by multiplying the principal amount by the interest rate and the time period, then dividing by 100. This gives the amount of interest earned or paid over the specified time.

## How to Run the Script

1. Make sure you have Bash installed on your system
2. Download or clone the `simple-interest.sh` file
3. Open a terminal in the file's directory
4. Run the command: `bash simple-interest.sh`
5. Follow the prompts to enter Principal, Rate, and Time

## Example Calculation

If you enter:
- Principal: 1000
- Rate: 5
- Time: 2

The calculation will be: (1000 × 5 × 2) / 100 = 100

The script will output: `Simple Interest = 100` and `Total Amount = 1100`
